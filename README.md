## Deploy to GitHub Pages

This repository deploys as a project site at:

- `https://hwgprojects.github.io/schlopping`

Steps:

1. Ensure `homepage` in `package.json` is `https://hwgprojects.github.io/schlopping`.
2. Build the static export:

```bash
npm run predeploy
```

3. Publish to the `gh-pages` branch:

```bash
npm run deploy
```
